## 知识点

**基础**

+ Array 数组 - 按下标访问
+ Linked list 链表
+ Stack 栈 - 括号匹配、水槽接水
+ Queue 队列 - BFS、滑动窗口
+ HashTable 哈希表
+ Set、Map - HashMap、TreeMap
+ BinaryTree 二叉树 - 递归
+ BST 二叉搜索树
+ Search 搜索
+ Recursion 递归
+ DFS 深度优先搜索
+ BFS 广度优先搜索
+ Divide & Conquer 分治
+ Backtracking 回溯
+ Greedy 贪心
+ Binary Search 二分查找

**进阶**

+ Trie Tree 字典树
+ Disjoint Set 并查集
+ Bloom Filter 布隆过滤器（不能删除，只能清空）
+ LRU Cache（LFU、ARC） -> [cache基础知识](https://www.jianshu.com/p/8dee805a5c89)
